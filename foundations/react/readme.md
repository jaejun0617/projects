# React Foundations

이 디렉토리는 React를 배우면서
JavaScript에서 정리한 개념들이
어떻게 React의 구조와 설계로 이어지는지를 이해하기 위해 정리한 문서들입니다.

React 문법을 빠르게 사용하는 것보다,

- 상태(state)와 렌더링의 관계
- 컴포넌트는 왜 이렇게 나뉘는지
- 비동기, 에러, 데이터 흐름을 어떻게 다루는지

같은 개념적인 부분을 중심으로 기록하고 있습니다.

아직 React를 깊게 다루는 단계는 아니며,
JavaScript 학습 이후 자연스럽게 이어지는 사고 흐름을
정리하는 목적의 문서들입니다.
